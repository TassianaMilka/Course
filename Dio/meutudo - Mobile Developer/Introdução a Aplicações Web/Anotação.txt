# HTTP


O que é?

Um padronizado de comunicação entre duas aplicações que fazem aplicações conversarem de maneira padronizada.
Através de um padrão de requisitos(requests)e respostas(responses).


Comunicação

Http Request-O jeito padrão de solicitar coisas através do 
protocolo HTTP;

Http Response-O jeito padrão de enviar dados de resposta do 
protocolo HTTP;

Header-São configurações,observações ou adendos enviados
pelo servidor;

Content-O conteúdo de fato solicirado pelo usuário.
